## Features

- Create node, node cli project fast, easily
- Use [XO](https://github.com/xojs/xo) as linter
- Use [AVA](https://github.com/avajs/ava) as test runner

## Install

```
$ npm install -g pji
```

## Usage

```
$ pji
```
